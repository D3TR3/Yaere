## Yaere

Yaere is a full fledged Chat application built in React JS. 

Main features :
- Friends
- Reactions
- Tags
- Profile Picture
- Google SignIn
- Replying
- Mobile Responsiveness

Backend Built With Github Copilot.

## How to run :
npm start -- starts dev server

You'll have to import your own firebase config.
